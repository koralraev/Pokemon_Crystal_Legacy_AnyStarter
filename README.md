# Pokemon Crystal Legacy with a Gift NPC for any pokemon

I like not using the starter pokemons and setting myself a challenge of using mono-type teams for a playthrough. However, this can sometimes be troublesome if my planned "starter" pokemon is only available late in the normal game. 

This fork of Crystal Legacy adds the possibility to choose whichever pokemon you want (almost) from the start of the game by talking to an NPC in Elm's Lab.
The NPC will allow you to choose any pokemon available in the game (1-251), choose its level from 1 to 100, and whether or not it should be shiny. It is necessary to first obtain the pokedex from Oak at Mr. Pokemon's house and return to Elm and speak with the officer to name the rival, before any pokemon can be chosen. The NPC will check if you have the pokedex or not.

This mode is neither meant as a challenge or easy-mode. It simply enables you to play with any team you want. You can get a second starter pokemon, or start with a Kanto starter. You can start out with a team full of level 100 legendaries. You can complete the pokedex before battling Flakner. 

### Features

+ Gift NPC at Elm's Lab will give the player any requested pokemon, at any level, shiny or not.
